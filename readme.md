# BetterCanva 🖌️

**BetterCanva** is a premium, native desktop workstation for Canva. It transforms the standard Canva web experience into a high-performance, distraction-free environment packed with tools creators actually need.

## ✨ Core Features

### 🚀 BetterDash (Personalised Dashboard)
Jump straight into your workspace. Use **LucidSearch** to find recent projects or launch new designs in a few words. Access your launchpad, theme settings, and local upload management from one unified hub.

### 🎶 Focus Canvas
Get locked into your flow. Focus Canvas hides the distracting top navigation bar (Home, File, Share) while you work. Need them back? Simply hover your cursor at the top of the tab to reveal them dynamically.

### 🎨 Custom Themes
Canva's desktop app is visually limited. We’ve unlocked a library of dynamic themes including **Terminal Matrix**, **Canva Glow**, **Gaming Legacy RGB**, and **Celebration (with particle effects!)**, specifically tailored for your tab bar.

### ⬆️ Fast-er Uploads
Standard browser uploads are slow and prone to failure. With **Local Uploads**, just drag your files into the BetterCanva Uploads folder. The app handles the upload process in the background—even if the window is minimized.

### 🔍 LucidSearch
The ultimate command palette. Press **Ctrl + Alt + Space** anywhere in the app to search your entire design history or trigger workflow tools instantly.

## 🛠️ Built for Creators
BetterCanva is built using **Electron.js**, providing:
* **Export Version Tracking:** Automatic version control for all your downloaded design exports.
* **Zen Mode:** A distraction-free workspace environment.
* **Zero-Library Engine:** 3D physics, glassmorphism UI, and interactive animations built in pure, lightweight vanilla code.
